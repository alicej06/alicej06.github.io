Robins Cut
AI-powered film production dashboard for data-driven development decisions
GPT Integration ROI Prediction Full-Stack Python ML Regression Film & Media

Overview

Robins Cut is a full-stack AI dashboard built for film producers and development executives to make more data-informed decisions throughout the production pipeline — from script analysis to greenlight forecasting.

The platform combines an ROI prediction model (trained on historical box office and production data) with GPT-powered script generation and production planning tools.

Technical Approach

  • ROI prediction model: regression model trained on genre, budget, director/cast metadata, and release timing features to forecast box office performance
  • GPT integration for script generation: constrained prompting to generate scene outlines, dialogue rewrites, and logline variations aligned to genre and tone inputs
  • Production planning module: schedule and budget estimation tools using historical production data
  • Full-stack implementation with dashboard UI, user authentication, and project management features

Key Features

  • Greenlight score — composite AI-generated assessment of commercial and creative viability
  • Script analyzer: genre classification, tone mapping, and comparable title matching
  • Budget sensitivity analysis: how changes in cast, format, and release window affect projected ROI
  • GPT scene generator for rapid script iteration and development exploration
