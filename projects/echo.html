Overview

Echo is a real-time sign language translation system built around an EMG-based armband that reads muscle signals from the hand and forearm, translating gestures into text and audio output via a companion mobile app.

Led a 6-member cross-functional team to design, build, and validate the system from hardware integration to model deployment — winning 1st place at the USC Marshall AI Summit 2026.

Technical Approach

  • Built a self-collected dataset of 1,900 labeled EMG samples across 10 ASL gestures using a Myo armband
  • Implemented a DTW-SVM (Dynamic Time Warping + Support Vector Machine) classification pipeline for temporal gesture recognition
  • Engineered preprocessing pipeline: bandpass filtering, RMS feature extraction, windowing for real-time inference
  • Integrated with a mobile app for live translation output with sub-100ms latency

Validation & Impact

  • Conducted 14 user interviews with Deaf and hard-of-hearing individuals to validate use cases and UX
  • Partnered with disability centers at USC for user testing and feedback integration
  • Achieved 99.5% classification accuracy on held-out test set
  • Presented to 200+ attendees at USC Marshall AI Summit; awarded 1st place
